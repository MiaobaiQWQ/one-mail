OneMail

本地优先的多账号桌面邮件客户端。把 Gmail、Outlook、QQ、163 和自定义 IMAP 邮箱放进一个安静、可备份、重视隐私的工作台。

为桌面邮件工作流设计

少一点云端黑盒，多一点本地掌控

OneMail 关注每天真的会用到的邮件动作：同步、筛选、阅读、回复、下载附件和备份数据。

01

多账号聚合

支持 Gmail、Outlook、163、QQ 和自定义 IMAP，账号栏展示未读数、同步状态和常用操作。

02

快速筛选

未读、有附件、星标、今日等标签可组合使用，让收件箱从信息洪流回到可处理列表。

03

按需加载正文

列表先同步邮件头，打开邮件时再加载正文，减少启动负担，也保留更清爽的阅读节奏。

04

Gmail 风格写信

支持写信、回复、回复全部、转发、抄送/密送、富文本、附件和草稿保存。

05

附件下载

正文底部展示附件表格，保存前只保留元数据，点击下载时再写入本地文件。

06

同步策略

可配置同步间隔、缓存窗口和外部图片策略，适合长期运行的桌面工作环境。

数据与安全

邮件留在你的电脑里

OneMail 使用本地 SQLite 缓存账号、邮件头、正文、附件元数据、搜索索引和设置。邮箱密码或授权码会使用 AES-256-GCM 加密后保存，HTML 邮件默认拦截远程图片和外部资源。

阅读安全说明
加密凭据 AES-256-GCM
本地数据库 SQLite
远程资源 默认阻止
数据迁移 SQL 备份
